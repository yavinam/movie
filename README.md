<h2>🛠️ Installation Steps:</h2>

<p>1. Install all dependencies</p>

```
npm install | yarn install
```

<p>2. Run application</p>

```
npm run dev | yarn dev
```

<p>3. Write your own environment values</p>

```
.env
```
